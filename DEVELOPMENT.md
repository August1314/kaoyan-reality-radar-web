## 技术架构

### 目录结构

```
src/
├── components/       # UI 组件
│   ├── SearchInput.tsx      # 智能搜索框
│   ├── SearchHistory.tsx    # 搜索历史
│   ├── ShareButton.tsx      # 分享按钮
│   ├── ThemeToggle.tsx      # 主题切换
│   ├── RadarChart.tsx       # 雷达图 SVG
│   ├── FailureCard.tsx      # 失败经验卡片
│   └── ...
├── hooks/            # 自定义 Hooks
│   ├── useSearchHistory.ts  # 搜索历史管理
│   └── useTheme.ts          # 主题管理
├── pages/            # 页面组件
├── data/             # 静态数据
├── lib/              # 工具函数
└── App.css           # 全局样式
```

### 关键断点

- `900px`: 平板适配
- `640px`: 小屏适配
- `480px`: 移动端适配

### CSS 变量

```css
:root {
  --color-bg: #ffffff;
  --color-text: #1f2937;
  --color-primary: #5b21b6;
  /* ... */
}

:root.dark {
  --color-bg: #0f172a;
  --color-text: #f1f5f9;
  --color-primary: #a78bfa;
  /* ... */
}
```

## 开发流程

1. **创建分支**: `git checkout -b feat/xxx`
2. **开发**: `npm run dev`
3. **检查**: `npm run lint && npm run build`
4. **提交**: `git commit -m "feat: xxx"`
5. **推送**: `git push origin feat/xxx`
6. **验证**: 检查 Vercel Preview
7. **合并**: 合并到 main，触发生产部署
