# apple.com 映射 Tokens

采集时间：2026-04-13  
结论类型：结构映射用，不是品牌复刻。

## 颜色方向

- 背景：非常浅的中性灰白
- 内容底：纯白或接近纯白
- 主文字：接近黑色但非纯黑
- 次级文字：中灰
- 强调色：克制蓝，用于链接和少量操作
- 边框：低对比细边框

## 版式方向

- 主容器更宽，首页允许更接近 full-bleed 的布局
- 卡片半径偏大，但阴影很轻
- 段落之间靠留白切换，不靠颜色乱跳
- feature panel 比普通卡片更大，信息更少

## 字体层级

- H1：极强，但不能压到正文阅读
- H2：用于 feature panel 和大段标题
- 正文：短句为主，不使用连续长说明
- 元信息：尽量单行，不堆小字

## 组件映射

- Apple global nav -> `PageRouteBar` 的轻量壳层
- promo bar -> 首页顶部单行信息带
- hero panel -> 首页首屏搜索区
- full-width product section -> 首页 feature panels
- 2-up tiles -> 首页次级入口区
- compact footer CTA -> 首页底部投稿与边界说明
